Sécurité & Mentions Légales

Dernière mise à jour de conformité effectuée en juillet 2026.

1. Édition et Propriété Intellectuelle

Cette application web est la propriété exclusive de Gabriel VOYER. L'intégralité du code source applicatif front-end, les feuilles de style structurelles et l'agencement graphique constituent une œuvre de l'esprit protégée par les lois en vigueur sur la propriété intellectuelle.

2. Traitement Secrétisé des Données (RGPD)

PostLink applique une politique stricte de confidentialité *Zero-Server-Storage*. **Aucune des données textuelles saisies dans l'éditeur n'est stockée à distance ou revendue.** Les données sauvegardées dans l'historique restent exclusivement cloisonnées dans la mémoire locale de votre navigateur internet (mécanisme LocalStorage). L'appel au modèle d'intelligence artificielle est éphémère et chiffré.

3. Architecture Cloud & API

Les requêtes d'analyse cognitive et de pré-rédaction sont acheminées via une passerelle d'API découplée et sécurisée. Le moteur sous-jacent est orchestré sur des serveurs sécurisés pour empêcher toute fuite ou exposition de vos clés d'accès.
